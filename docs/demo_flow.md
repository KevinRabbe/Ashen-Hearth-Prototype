# Demo Flow

## Target length

20–40 minutes.

The demo should be short enough that friends can replay it often and give feedback without a huge time commitment.

## Main goal

Prove that the following loop is fun:

```text
Fight -> Loot -> Build -> Upgrade -> Boss -> Trophy -> Home -> Sleep -> Next Day
```

## Phase 1: Broken camp

Estimated time: 3–5 minutes.

Player starts at a ruined camp on the edge of the Cursed Forest Ruins.

Starting objects:

- broken workbench
- broken armory station
- campfire
- empty trophy mount
- player bed area
- damaged chest
- ruined walls/floors

Player learns:

- movement
- interaction
- basic building placement
- basic repair/crafting interaction

Objective:

> Repair the workbench and prepare the camp.

## Phase 2: Find Ash

Estimated time: optional within first 5–10 minutes.

Ash is slightly hidden near the camp or forest edge.

Possible hiding spots:

- under broken cart
- behind collapsed wall
- near ruined fireplace
- inside hollow tree
- behind tall grass

Soft hint:

> You hear a faint meow near the old camp.

Ash should be easy to miss if players rush, but easy to find if they explore.

Objective after discovery:

> Gain Ash's trust.

Trust:

```text
Ash Trust: 0/3
```

Pet/feed interactions raise trust. At 3/3 Ash moves to the camp, but she does not have a proper bed yet.

## Phase 3: First forest fight

Estimated time: 5–10 minutes.

Player enters the nearby forest path.

Encounters:

- Bone Thralls
- Corrupted Wolf
- basic resource nodes
- 1 small loot chest

Player learns:

- light attack
- heavy attack
- blocking/dodging
- loot pickup

Objective:

> Gather materials to repair the armory.

## Phase 4: First upgrade

Estimated time: 3–5 minutes.

Player returns to camp.

Actions:

- repair armory
- craft/upgrade sword slightly
- place a few base pieces
- rest near bed/campfire

Objective:

> Upgrade your weapon and prepare for the ruins.

## Phase 5: Ruins run

Estimated time: 10–15 minutes.

Player enters the cursed ruins.

Encounters:

- Bone Thralls
- Corrupted Wolf variant or ambush
- Gravebound Knight elite
- 2 loot chests
- optional side room
- boss shrine gate

Objective:

> Find the Cursed Iron Core and open the Bone Warden shrine.

## Phase 6: Mini-boss fight

Estimated time: 5–10 minutes.

Boss:

> The Bone Warden

Boss rewards:

- Bone Warden Trophy
- Cat Bed Blueprint
- Cursed Iron Core
- one random building blueprint

Objective:

> Defeat the Bone Warden and return home.

## Phase 7: Home payoff

Estimated time: 3–5 minutes.

Player returns to camp.

Actions:

- mount Bone Warden Trophy
- build Ash's Cat Bed
- watch Ash sleep/settle in
- sleep in player bed

Objective:

> Rest until morning.

## Phase 8: Day 2 begins

When players sleep:

- fade out
- show short summary
- reset enemies/resources outside
- keep base, unlocks, trophy, and Ash
- wake players in their beds

Example summary:

```text
Day 2 Begins
Bone Warden defeated
Blueprints found: 2
Ash has settled into the camp
Stronghold comfort increased
```

## Replay hooks

Even in the demo, include a few reasons to replay:

- rare blueprint drops
- different base layout
- faster boss kill
- missed Ash discovery
- better camp design

## Success signs

The demo is working if players say:

- “Let's run it again.”
- “I want to build the base differently.”
- “Where does that blueprint drop?”
- “Ash makes this feel like home.”
- “The boss reward felt worth bringing home.”
